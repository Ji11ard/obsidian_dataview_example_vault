# List files or metadata starting with a certain letter

## Basic 

For filenames:
```dataview
LIST
FROM "10 Example Data"
WHERE file.name[0] = "A"
```

For metadata fields:

```dataview
LIST
FROM "10 Example Data"
WHERE author[0] = "B"
```
